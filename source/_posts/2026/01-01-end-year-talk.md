# 2025 年终杂谈：好像今年，也没那么糟

十几个小时前，2025年悄然走完了全程，我们这些平凡人被时光轻轻推入2026年。新岁首日，空气里都浸着几分微醺的喜悦。忙完期末考试与期末汇报，终于能抽出身做些真正想做的事——写博客、敲代码、健身、读论文，一切似乎与往日无差。尽管仍在求职路上跋涉，前方依旧横亘着不少挑战与阻碍，但我总算沉下心来了。**船到桥头自然直，有时不由船主掌控，只随水流自在前行。**

看着QQ、B站、网易云纷纷推送年终总结，我想，我也该用一篇博客为这一年做次梳理——盘点做过的事、写过的文、走过的路。

## 博客写作：在记录与分享中前行

2025年，我一共写下了**21篇博客**，这个数字打破了2023与2024两年的总和记录。这份成果让我略感欣慰，至少在输出与记录这件事上，我比往年坚持得更久、做得更好。

这些文章的主题涉猎颇广：

- **技术深度**：《Llama1-3 🦙：从一道美团大模型面试题讲起》《DeepSeek 🐬：国产大模型之光？它用到了哪些关键技术？》《从零入门 CUDA 编程？🦴 看这篇就够了！》及其续篇
- **知识梳理**：《一篇博客带你回顾前端发展的 20 年🫕》《老生常谈🗣：“进程、线程、协程”的相关概念汇总辨析》《从集合论到位运算，常见位运算技巧分类总结！》
- **实用工具**：《使用 rofi 搭建好看的电源管理界面🐘》《如何写更好的 git 提交信息 (git message)》《深入理解 Linux 字体配置：我的 fontconfig 实践 🧭》
- **算法与系统**：《BFS、DFS、UCS、A\*：常用的搜索算法简介 🔎》《你不得不学的线段树 (segment tree) 基础 🌳》《Kafka 技术在 B 站 📺 的探索与实现》
- **行业思考**：《最流行 🐦 的 6 种 API 架构风格》《什么是 MCP？模型上下文协议？》《一块旧时代的墓志铭：大数据、区块链、元宇宙》
- **心路随笔**：《六月份杂谈：西西弗斯、焦虑、迷茫、坚守》《九月份杂谈：秋招滑铁卢、武大Offer、躺平加拖延》

涉猎虽广，但回头细想，内容多是技术干货，少见生活里的趣味点滴。我还没能抽出勇气或闲暇，写下那些同样热爱的动漫、游戏与音乐；论文阅读笔记的产出也有所减少，可能是精力被学习与求职任务分流后的必然结果。希望来年，我的博客能更完整地映照出“我”的全貌，而非仅仅是一部“技术记录仪”。

## 科研学习：在迷雾中摸索方向

这一年，我初步锚定了科研兴趣方向——**高效推理与模型压缩**。围绕这个方向，我陆续研读了约**15篇**相关论文，核心聚焦于**剪枝、量化与知识蒸馏**三大领域。纸上得来终觉浅，理论积累的过程固然充实，却也藏着强烈的焦虑：我怕毕业设计最终流于表面，不得不“注水”应付。

![论文阅读](read.jpg)

一个明显的遗憾是，我**没能将读论文时的心得与笔记系统整理成博客**。那些转瞬即逝的灵感、悬而未决的困惑、灵光乍现的联想，大多散落在凌乱的文档与书页批注里。这也让我明白：输入固然重要，但经过消化、重构后的输出，才是知识真正内化的开始。科研之路道阻且长，愿2026年的自己能更从容、更扎实地走下去。

## 代码耕耘：在 GitHub 上留下痕迹

我的2025年GitHub「代码年度报告」，交出了一份还算满意的答卷：

- **持续活跃**：最长连续**201天**提交代码，全年总贡献**756次**，周一果然是我的“代码爆发日”
- **语言主力**：**JavaScript**依旧稳坐最常用语言的宝座
- **项目探索**：新发现**15个**感兴趣的项目并点亮Star
- **协作贡献**：成功提交并合并**11个Pull Request**，切实参与到开源社区的互动中
- **问题处理**：今年重心在新功能开发，关闭的Issue数为0

这串数字背后，是无数个调试代码、学习新框架、尝试新工具的日常。GitHub上的绿格子，像一块被悉心耕耘的数字农田，清晰地刻下了我的节奏。这份专注输出的踏实，让我倍感充实。

## 身体与生活：在失衡与恢复中循环

健身是今年起伏最明显的坚持：

- **春季高光**：选修健美健身课，拿到92分的好成绩，体重也成功降到65kg，肌肉线条清晰可见
- **暑期荒废**：被保研筹备与秋招冲刺占满精力，训练几乎完全中断
- **秋季重启**：9月加入健美健身协会，重新拾起训练计划，甚至有过单日跑完10公里的突破
- **冬季挫折**：11月尝试高强度训练导致手肘劳损，整个12月都被迫暂停休养

今年年终的时候，我和几个健身协会的同好一起到健身房跨年。这是我人生第一个在健身房度过的跨年了，哈哈。希望新的一年里，我能够风雨无阻地完成健身计划，不被节假日和各种 ddl 牵着鼻子走，安安心心地打磨自己的肉体，追求完美的形体。**我了解到“训迹”这个 app 可以免费记录你当天的健身打卡，我想后面我会使用它吧。**

## 精神食粮：在屏幕里构建另一个世界

Bilibili堪称我2025年的“时间黑洞”，累计观看**1117小时**，超过95%的用户，成了我每日不可或缺的陪伴：

- **最爱UP主**：@吃蛋挞的折棒（1672分钟）、@四郎讲棋（1187分钟）、@戴宗如何（596分钟）
- **兴趣领域**：影视、知识、游戏、动画、动物、小剧场……我的爱好还是一如既往地驳杂
- **年度名场面**：11月16日，竟一口气刷完338个视频
- **电子榨菜**：261顿午饭，都是伴着《红楼梦》的剧情咽下的。四大名著里，《水浒传》（955分钟）、《三国演义》（783分钟）、《红楼梦》（762分钟）已重温完毕，只差《西游记》就能集齐“全套”

动画与漫画的世界，同样给了我温暖的慰藉：

- **新番追踪**：《小城日常》《胆大党》《从前有勇者》《东岛丹三郎想当假面骑士》
- **老番重温**：《间谍过家家》《动物新世代》《刺客伍六七》《火星西路七号》
- **漫画追更**：《谢了你啊异世界》《J 和 M》《无能的奈奈》《异世界舅舅》等十余部，甚至翻出《三眼哮天录》《偷星九月天》这些国产老漫重温旧时光

游戏方面，我有过两次深度沉浸的体验：

- **6月**：沉迷《动物迷城》，耗时151小时；探索《苏丹的游戏》，投入94小时
- **9月**：在《猛兽派对》中结识新朋友，共同游玩35小时

这些屏幕里的故事、对局与角色，拼凑成我2025年最重要的快乐源泉与放松空间。

## 关于“兽”与未来：一个小愿望

今年没能参加任何一场兽聚，算是个小小的遗憾。但我也完成了一件期待已久的事——**约了一张属于自己的兽设画稿**。看着画里的自己，感觉离那个毛茸茸的梦想社区，又近了一大步。

![2025年约的大作(by竹桉)](myoc.jpg)

希望2026年，能约到更多合心意的画稿，慢慢拼成一面满是故事的“兽设照片墙”；更希望能攒钱买到心心念念的兽装，走进线下兽聚，认识更多志同道合的朋友，真正融入那个充满创造力与温情的圈子。

## 尾声：好像今年，也没那么糟

敲下这些文字时，2026年的时针已悄然转动了许久。回望2025，这一年满是焦虑（求职的迷茫、毕业的压力）、挫折（伤病的困扰、计划的中断）与不确定性；但与此同时，它也藏着太多创造（21篇博客的产出）、坚持（756次代码提交）、探索（新的技术领域与兴趣圈层），以及那些微小却扎实的成就（健身的进步、第一幅兽设画稿的达成）。

是的，**好像今年，也没那么糟**。它不完美，却足够真实；它有挣扎，更有生长。这是我在迷茫中摸索方向、在压力下努力喘息、在热爱里汲取能量的一年。

水流或急或缓，桥头终会抵达；前路或明或暗，脚步终会踏实。带着2025年的收获与遗憾，我认真走进2026年。新的一年，愿我能更从容地写代码、更健康地生活、更勇敢地分享热爱，也愿我能顺利找到那座属于自己的“桥”。⛵
